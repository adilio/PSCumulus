# Slide Build Notes

This deck should follow the guardrails in `docs/concepts/strategy.md`:

- one message per slide
- six objects max
- phrase-heavy slides over sentence-heavy slides
- use Summit theme utilities instead of ad hoc styles

Before exporting:

1. Re-check live factual claims.
2. Trim any slide that feels like speaker notes.
3. Verify code samples are readable from the back of the room.
